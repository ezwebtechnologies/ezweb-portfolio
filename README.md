# EZWeb

Marketing site for **EZWeb**—integration, API, and automation solutions.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Set `NEXT_PUBLIC_SITE_URL` to your production origin (used for canonical and Open Graph URLs).

## Stack

Next.js (App Router), TypeScript, Tailwind CSS.
